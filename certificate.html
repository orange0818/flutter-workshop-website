Name matching

    Select one of the suggested names to download your exact certificate file.

    Tip: The download will only proceed when the entered name exactly matches a suggested student from the list. This prevents spelling mismatch errors.

    How this portal works

    • Use the left side to check whether a certificate code is valid.
    • Use the right side to select your exact student name and download the certificate image.
    • If a certificate exists for the selected name, the download button becomes active.
    • If the code is invalid, the warning message will explain that the certificate was not found.
